# WechatHook

### WeChat  Hook 
本项目是一个基于 Python 的实用脚本，通过 pymem 库动态修改运行中微信（WeChat.exe）的内存数据。主要用于解决部分旧版本微信登录时提示“版本过低”的问题。

### 📌 功能原理
脚本会自动定位微信的核心模块 WeChatWin.dll，在内存中搜索特定的版本标识符位字节（特征码 63090C39），并将其替换为目标版本号（F254160E）。这种修改仅在内存中生效，不会破坏本地文件。
### 使用步骤
直接右键以管理员运行就行
